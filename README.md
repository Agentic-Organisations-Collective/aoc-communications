# aoc-communications

Public-facing communication of the Agentic Organisations Collective, separated
from the legal and member domains. It publishes outward-facing communication
produced from governed source material.

## Packages

- `event-teasers` — public teasers and short previews for upcoming events
- `announcements` — official public announcements
- `press` — press-facing material and media communication
- `website-content` — editorial content published on the public website
- `social-posts` — short-form posts for social channels

## Governance

Public read, contributor write. This repo may consume released content from
governance, but not the other way around.
